Developing 
==========

.. note:: If you would like to contact me please view the :ref:`contact` page
    
Contribute To The Project
-------------------------
If you wish to contribute to this project please feel free to try either of these methods.

Pull Request
~~~~~~~~~~~~
If you would like to contribute dirctly to the main branch please make a pull request and I will review your changes.

Branch
~~~~~~
You are also welcome to create a new branch and continue working on the new brach. If you would like to merge the new branch into main please create a mrege request?

Fork
~~~~
Finally, if you want to work in a repository of your own please fork the repository.

Bug Reporting
~~~~~~~~~~~~~~~~
If you would like me to fix a bug/issue please feel free to create a new issue and I will get to it in a new release.
